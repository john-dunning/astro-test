mFactory
    
 

Another addition to mTropolis 2.0 were what we called mPacks. These libraries of pre-built components could be dropped into a user’s project. Instead of having to create a complicated widget like a scrollbar from scratch, a user could simply grab one from the Scrollbar mPack.

Since these components were built from standard mTropolis elements and modifiers, a user could easily customize them as needed. They also provided useful authoring examples.

I developed most of these mPack components, since I needed them to build the interfaces of the plug-in tools. I also created an mPack Guide tool, which documented each component and showed a working sample, as seen here.
